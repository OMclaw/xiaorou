## 更新内容

### 1. 三种生图模式定义

#### 1️⃣ 场景生图 (selfie-scene)
- **定义**: 根据场景描述用小柔默认头像生成
- **触发条件**: 纯文字场景描述，无图片
- **关键词**: `发张自拍`、`想要一张`、`生成一张`、`来一张`、`穿`、`穿搭`、`在...里/前/下`、`自拍`、`照片`、`图片`、`发张`、`看看你`、`生成`、`想要`、`场景`
- **示例**:
  - "发张自拍，在海边看日落"
  - "想要一张在咖啡厅看书的照片"
  - "生成一张在樱花树下的场景"
  - "来一张穿汉服在古城的照片"

#### 2️⃣ 参考生图 (selfie-reference)
- **定义**: 基于参考图识别后用小柔头像生成
- **触发条件**: 有图片 + 参考类关键词
- **关键词**: `参考`、`模仿`、`照着`、`学这张`、`类似的`、`同样的`、`照这个`、`按这个`、`生成一张`、`来一张`
- **示例**:
  - "参考这张图生成一张" + [图片]
  - "模仿这个场景来一张" + [图片]
  - "照这个样子生成" + [图片]
  - "生成一张类似的" + [图片]
- **流程**:
  1. 分析参考图 → 提取场景、姿势、服装、光线等描述（忽略人脸）
  2. 使用小柔头像作为图生图的输入
  3. 双模型并发生成（wan2.7-image + qwen-image-2.0-pro）

#### 3️⃣ 换脸生图 (face-swap)
- **定义**: 用用户提供的图片作为图 1，小柔默认头像作为图 2，进行换脸
- **触发条件**: 有图片 + 换脸类关键词（优先级最高）
- **关键词**: `换脸`、`换我的脸`、`把脸换成`、`用我的脸`、`face swap`
- **示例**:
  - "换脸" + [图片]
  - "把脸换成小柔" + [图片]
  - "用我的脸，换成小柔" + [图片]
  - "face swap" + [图片]
- **流程**:
  1. 使用用户提供的图片作为图 1（目标场景）
  2. 使用小柔默认头像作为图 2（脸部来源）
  3. 分析目标图场景（不含脸部）
  4. 生成精准换脸方案：小柔的脸 + 场景精确还原
  5. 双模型并发生成（wan2.7-image + qwen-image-2.0-pro）

### 3. 模式优先级

```
1. 换脸模式 (face-swap)      - 最高优先级，匹配"换脸"类关键词 + 有图片
2. 视频模式 (video)          - 匹配"生成视频"类关键词
3. 语音模式 (voice)          - 匹配"发语音"类关键词
4. 参考生图 (selfie-reference) - 有图片 + 参考类关键词
5. 场景生图 (selfie-scene)    - 有场景描述关键词（无图或有图无参考词）
6. 聊天模式 (chat)           - 默认模式
```

## 技术细节

### 正则表达式修复
**问题**: `grep -qiE "关键词 1 | 关键词 2"` 中的 `|` 前后有空格会导致匹配失败

**原因**: 在扩展正则表达式中，空格会被当作模式的一部分

**修复**: `grep -qiE "关键词 1|关键词 2"` - 移除 `|` 前后的空格

### 模式检测逻辑
```bash
# 换脸模式（优先级最高）
if echo "$input" | grep -qiE "换脸 | 换我的脸 | 把脸换成 | 用我的脸|face.?swap"; then
  if [ -n "$has_image" ]; then
    echo "face-swap"
    return
  fi
fi

# 参考生图模式（需要图片）
if [ -n "$has_image" ]; then
  if echo "$input" | grep -qiE "参考 | 模仿 | 照著 | 学这张 | 类似的 | 同样的 | 照这个 | 按这个 | 生成一张 | 来一张"; then
    echo "selfie-reference"
    return
  fi
fi

# 场景生图模式
if echo "$input" | grep -qiE "照片 | 图片 | 自拍 | 发张 | 看看你 | 穿 | 穿搭 | 生成 | 来一张 | 想要 | 场景 | 在.*里 | 在.*前 | 在.*下"; then
  if [ -n "$has_image" ]; then
    echo "selfie-reference"  # 有图无参考词 → 参考生图
  else
    echo "selfie-scene"      # 纯文字 → 场景生图
  fi
  return
fi
```
