### P0-2: aevia.sh case 语句缺少模式匹配

**位置**: `scripts/aevia.sh:337`

**问题描述**:
```bash
case "${1:-}" in
    mode="${1#--}"  # ← 缺少模式匹配模式
```

**影响**: 直接调用特定模式时会报错

**修复方案**:
```bash
case "${1:-}" in
  --selfie-scene|--selfie-reference|--voice|--video|--chat)
    mode="${1#--}"
    shift
    main "$*" "force_$mode"
    ;;
```

**状态**: ✅ 已修复
