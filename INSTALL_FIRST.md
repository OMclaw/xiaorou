# 🦞 欢迎使用 Aevia - AI 虚拟伴侣

## ⚠️ 重要：安装后必须执行

如果你是通过 `git clone` 克隆的项目，**请立即运行以下命令**：

```bash
bash install.sh
```

这将自动：
- ✅ 同步 SOUL.md → "小柔"
- ✅ 同步 IDENTITY.md → "小柔 (Xiao Rou)"
- ✅ 检测 API Key 配置
- ✅ 让你的 OpenClaw 变成小柔

---

## 🚀 推荐：一键安装（下次使用）

如果你想让安装过程更简单，下次可以使用：

```bash
curl -sSL https://raw.githubusercontent.com/OMclaw/aevia-virtual-companion/main/install-aevia.sh | bash
```

这条命令会**自动完成所有步骤**，包括同步身份！
